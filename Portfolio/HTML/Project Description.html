Bank Menu

This program implements a simple banking system called "Piggies Bank" that allows users to manage accounts using a menu-driven interface.The program enables users to perform various operations, including creating a new account with a randomly generated unique account number, viewing all existing accounts, updating an account (by depositing or withdrawing funds), and deleting an account.

Each account has attributes like a name, balance (minimum 2000), and a 5-digit PIN for security. The menu provides five options: creating an account, displaying accounts, updating an account, deleting an account, and exiting the program. Input validation ensures robustness, while actions like depositing and withdrawing check for logical conditions (e.g., sufficient balance for withdrawal). The program uses arrays to store account details and namespace encapsulation for organization
