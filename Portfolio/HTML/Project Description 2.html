Cake Order Menu

This program is a menu-driven application for a fictional bakery, "PURRfect Pastries," allowing customers to interactively place and manage their cake orders. The menu offers options to order cakes, add layers to cakes, delete an order, and generate a bill for the ordered items. Customers can select cakes from a predefined menu with prices and add layer options for customization.

Each order is managed using arrays, with options to delete specific items or view the current bill. Input validation ensures smooth operation, and detailed outputs summarize the customer's choices and total cost. The program is structured to handle up to 10 cake and layer orders efficiently, enhancing the user experience with a personalized touch.
